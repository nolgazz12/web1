Abstract This specification defines the 5t h major revision of the core language of the World Wide Web: the Hypertext Markup Language (HTML). In this version, new features are introduced to help Web application authors, new elements are introduced based on research into prevailing authoring practices, and special attention has been given to definingclear conformance criteria for user agents in an effort to improve interoperability.
